Загрузка курсов валют НБРБ

Возникла нужда воспользоваться Web-сервисом НБРБ для загрузки курса в 1С:Предприятие 8.

Получился такой пример:

ОпределениеСервисаНБРБ = Новый WSОпределения("http://www.nbrb.by/Services/ExRates.asmx?WSDL");
СервисНБРБ = Новый WSПрокси(ОпределениеСервисаНБРБ, "http://www.nbrb.by/", "ExRates", "ExRatesSoap");

ТипПараметраДатаКурса = СервисНБРБ.ФабрикаXDTO.Тип("http://www.nbrb.by/", "ExRatesDaily");

ДатаКурса = СервисНБРБ.ФабрикаXDTO.Создать(ТипПараметраДатаКурса);
ДатаКурса.onDate = ТекущаяДата();

КурсыВалютНаДату = СервисНБРБ.ExRatesDaily(ДатаКурса);
СписокКурсовВалют = КурсыВалютНаДату.ExRatesDailyResult.diffgram.NewDataSet.DailyExRatesOnDate;

Объект.КурсыВалют.Очистить();
Для Каждого ИнформацияОКурсеВалюты Из СписокКурсовВалют Цикл
 КурсВалюты = Объект.КурсыВалют.Добавить();
 ЗаполнитьЗначенияСвойств(КурсВалюты, ИнформацияОКурсеВалюты);
КонецЦикла;

Скачивание зависимостей пакетов yum

03 Jul 2016, 00:00

Столкнулся с необходимостью установки докера на целевой хост без доступа в интернет.

Пазвернул посредство вагранта CentOS 7 и получил все пакеты (естественно прописав репозиторий докера):

$ sudo yum install --downloadonly --downloaddir=docker docker-engine

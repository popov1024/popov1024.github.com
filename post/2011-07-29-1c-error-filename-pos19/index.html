1С error - Для полей, начиная с FileName, не хватило значений (pos=19)

Ошибка проявляется на 1С:предприятие 8.1.15.14

В процессе обновления информационной базы произошла критическая ошибка.
по причине:
Ошибка SDBL:
Для полей, начиная с FileName, не хватило значений (pos=19)

Вариант решения — очистить таблицы изменений конфигурации:

TRUNCATE TABLE _ConfigChangeRec
TRUNCATE TABLE _ConfigChangeRec_ExtProps

docker compose tip0

11 Jun 2017, 22:08
docker / docker-compose / tips

Docker-compose по умолчанию использует в качестве имени проекта имя каталога и создает контейнеры видв folder_bar_1.

Изменить имя проекта можно с помощью параметра -p или, что удобнее всего, указать имя проекта в .env:

COMPOSE_PROJECT_NAME=foo

Имена контейнеров тогда всегда будут foo_bar_1.

TeamCity in Linux

November 13, 2013 TeamCity, Jetbraince, Linux, CentOS Estimated reading time: ~1 minutes

Установка TeamCity для Linux освещена не плохо, но как-то смазан процесс создания автостарта данного сервиса. В моем случае интересен только сам сервер (без сборщика-агента) и работает все на CentOS 5.

Отдельный пользователь в системе:

sudo /usr/sbin/useradd teamcity
sudo passwd teamcity

Содержимое стартового скрипта /etc/init.d/teamcity

#!/bin/bash
#
# teamcity      TeamCity Continuous Integration Server
#
# chkconfig: 345 80 30
# description: TeamCity is a continuous Integration Server
# processname: teamcity
# config: /etc/sysconfig/teamcity
#
# TeamCity Linux service controller script
case "$1" in
    start)
        su - teamcity -c "cd /opt/jetbrains/teamcity/bin/ ; ./teamcity-server.sh start"
        ;;
    stop)
        su - teamcity -c "cd /opt/jetbrains/teamcity/bin/ ; ./shutdown.sh"
        ;;
    *)
        echo "Usage: $0 {start|stop}"
        exit 1
        ;;
esac

Регистрация:

sudo /sbin/chkconfig --add teamcity
